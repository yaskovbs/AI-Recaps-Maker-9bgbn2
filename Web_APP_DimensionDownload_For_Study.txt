​🚀 אפשרויות התקנה (Installation)
​אפשרות 1: התקנה מהירה עם Docker (מומלץ) 🐳
​הדרך הקלה ביותר שמתקינה את כל התלויות (Go, yt-dlp, AI models) אוטומטית:


docker-compose up --build


אפשרות 2: התקנה ידנית (Manual Setup) 🛠️
​Backend (Go): התקן את Go וודא ש-yt-dlp מותקן במערכת.
​Frontend (React): הרץ npm install בתיקיית ה-client.
​Database: הגדר פרויקט ב-Supabase וחבר את ה-URL והמפתח לקובץ ה-.env.
​🛠️ זרימת עבודה (System Workflow)
​איסוף: שימוש ב-YouTube API (בלולאת Pagination) לאיסוף IDs ושמירה ב-Supabase.
​עיבוד: הורדה דרך yt-dlp וניתוח תוכן ע"י Gemini.
​תלת-מימד: יצירת מפת עומק (אופציונלי) והמרת הוידאו.
​ניהול: שעון עצר ל-44 יום על כל קובץ באחסון.

🌟 תכונות עיקריות (Key Features)
 * BYOK (Bring Your Own Key): שליטה מלאה בעלויות. המשתמש מזין מפתחות API אישיים עבור YouTube ו-Google Gemini. 🔑
 * Smart Playlist Fetching: איסוף אוטומטי של פלייליסטים שלמים עם סנכרון בזמן אמת ל-Supabase. 🔄
 * AI 3D Processing: המרת וידאו רגיל (2D) לתלת-מימד (Anaglyph) באמצעות מודלי עומק מתקדמים. ניתן לכיבוי לחיסכון במשאבים. 🕶️
 * Temporary Cloud Storage: אחסון הקבצים המעובדים למשך 44 ימים בלבד.
 * Live Countdown Timer: המשתמש יראה ליד כל קובץ שעון עצר המראה בדיוק כמה זמן נותר עד למחיקתו האוטומטית מהשרת. ⏱️
